Getting Started

Installing Wordpress

A working version of WordPress CMS must be installed before you can install rx-theme theme. If you need help installing WordPress, follow the tutorials

Installation

Download theme

Download rx-theme theme files from Your account ThemeForest > Downloads tab:

Installing rx-theme

Once you've downloaded the installation file on ThemeForest, extract it and locate a file called rx-theme.zip. After unzipping the file downloaded from ThemeForest you should upload it to your WordPress installation:

Installing from WordPress admin

  1. Login to WordPress admin
  2. Go to Appearance > Themes and click Add New
  3. Click on "Choose File" browse the path to rx-theme themes and select rx-theme.zip
  4. Click on "Install Now" and start uploading
  5. Your theme will appear in the theme list
  6. After successful theme installation please click - Activate

Installing via FTP

  1. Using FTP client, login to the server where your WordPress website is hosted
  2. Using FTP client, navigate to the /wp-content/themes/ directory under your WordPress website's root directory
  3. Using FTP client, upload the Rx-Theme directory to themes directory on remote server
  4. Your theme will appear in the theme list
  5. After successful theme installation please click - Activate

Theme wizard installation

Rx-theme comes with installation wizard plugin Rx Theme Wizard. All that you need is to install recommended plugin after theme activation(you can find the service notice in the top panel of WordPress dashboard).

After successful installation plugin will activate automatically. Rx Theme Wizard can install all required plugins, selected skin, and also import demo data for your theme. All that you need is to start install (you can find the service notice with "Start Install" button in the top panel of WordPress dashboard).

On the next step you will processed to theme wizard dashboard, here you can find your system information to be confident that all system indicators are compliance with the system installation requirements.

Demo Content and Plugin installation

In order to start demo content installation please click - Install Demo Content.

Skins

The next step is the skin choosing, after that you can click - "Start Install" button.

Plugins

The next step is to configure plugins which will be installed to your theme, after that click - "Next". During the installation process you can see the status bar and installation details for each plugin.

Demo Content

After successful plugin installation you have to choose in which way you want to install demo data on your future site.

Important! There are three available options, look closely to choose the more appropriate for you and click - "Next" button.

On the next step after choosing the appropriate option you will be prompted to enter your WordPress user password in order to process sample data importing. Here you can see the progress and completion status.

After the sample data importing there will be regenerating thumbnails process starting.

When installation and importing process be finished you will be redirected to Congratulation page, from there you can start customize your theme or take a tour.
